

После изменений сохранить в GitHub
git add .
git commit -m "описание изменений"
git push


История коммитов:
git log --oneline


git switch main - переключиться в центаральгую ветку (main)
git switch dev - Перейти на ветку разработки (dev)

git switch -c dev - Перейти на новую ветку и создать её (dev)


## git pull — это команда, которая забирает последние изменения из удалённого репозитория (GitHub) 
и сразу объединяет их с твоим локальным проектом. (“Скачать свежую версию проекта с GitHub и обновить у себя”)
На самом деле это две команды вместе:

git fetch - Скачивает изменения с GitHub, но не применяет их.
git merge - Встраивает эти изменения в твой локальный код.

Отправить изменения в Гугл 
npm run build
clasp push
